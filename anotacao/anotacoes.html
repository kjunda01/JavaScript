console.log serve para escrever uma mensagem no console.
Exemplo: console.log("Olá, mundo!");Clique para copiar

window.alert serve para mostrar um alerta na tela, aparece um quadrinho em cima.
Exemplo: window.alert("Minha primeira mensagem!");Clique para copiar

window.confirm aparece uma janela de confirmação.
Exemplo: window.confirm("Está gostando de JS?");Clique para copiar

window.prompt aparece uma janela perguntando alguma coisa.
Exemplo: window.prompt("Digite seu nome: ");Clique para copiar
