Retorna string, para conseguir fazer o número é necessário converter a variável.

É possível fazer essa conversão utilizando:
Number.parseInt(n) para números inteiros ou o Number.parseFloat(n) para números float.

Existe também a possibilidade de utilizar apenas o Number(n) para deixar o JS decidir o tipo de resultado.
