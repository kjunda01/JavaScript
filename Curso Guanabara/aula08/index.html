Operadores (Parte2) - Curso JavaScript #08

Operadores relacionais

    5 > 2 -> true - MAIOR
    7 < 4 -> false - MENOR
    8 >= 8 -> true - MAIOR OU IGUAL
    9 <= 7 -> false - MENOR OU IGUAL
    5 == 5 -> true - IGUAL
    4 != 4 -> false - DIFERENTE

    5 == 5 -> true - 5 NÚMERO É IGUAL A 5 NÚMERO
    5 == '5' -> true - 5 NÚMERO É IGUAL A 5 STRING
    5 === '5' -> false - 5 NÚMERO NÁO É IDÊNTICO A 5 STRING
    5 === 5 -> true - 5 NÚMERO É IDÊNTICO A 5 NÚMERO

Operadores lógicos

    ! - NEGAÇÃO
    && - CONJUNÇÃO ('E', and)
    || - DISJUNÇÃO ('OU', or)

Operadores ternários

    ? - CONFIRMA PRIMEIRA OPÇÃO
    : - CONFIRMA SEGUNDA OPÇÃO

    let média = 6
    média > 7 ? 'APROVADO':'REPROVADO'
    Resposta: REPROVADO
