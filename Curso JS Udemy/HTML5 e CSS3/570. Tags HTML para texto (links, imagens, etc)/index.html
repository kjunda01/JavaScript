Título 1

Tags legais de se conhecer e seus significados.

Tags semânticas:

"p", para parágrafo.

"strong", deixa o texto com significado de forte, dar semântica, estado lígico.
"em", deixa o texto em ênfase, itálico.
"a", para fazer um link para um site.
"img" , para acrescentar imagens.
"del", Texto riscado semântico
"ins", Texto logo em seguida para "adicionar" o que foi excluído com o del
"small", Texto menor, com uma ênfase menor.

"sup", para palavras elevadas, tipo, 225

"sub", para colocar palavras abaixo, tipo H2O

"br" para quebra de linhas

"q", para citação, exemplo: Eu sou um estudante de ADS.

"blockquote", para citações com identação. Exemplo:
Olá, mundo! Esta é uma citação com identação.

O blockquote vai fora do parágrafo p, não pode estar dentro do p.

"hr", para usar uma linha de quebra.

"code", Quando quiser citar um código, usar a tag code. Exemplo:
console.log("este é um código, olá mundo!") 

                    "pre", para tags de texto preformatado, para manter os espaços e todo o texto.

                    Tipo assim, mantendo do jeito que está escrito.          Olá, mundo!

                    console.log("Olá, mundo!")

                    fim.

"div" para um bloco, separa o texto, cria um bloco, exemplo:

Ola, eu sou um bloco!
continuando, ele faz a quebra do texto, diferente do span.

Tags normais

"b", deixa o texto somente negrito, estado físico.
"i", deixa o texto em itálico, somente.
"s", Texto riscado sem sentido.
"u", Underline, underscore, sublinhado como tags físicas.

"span", é uma tag de container visual, inline, pode ser usada dentro do parágrafo.
